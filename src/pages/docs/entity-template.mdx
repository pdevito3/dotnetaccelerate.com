---
title: Entity Template
description: The template for the file used when adding new entities to a web API using the `add:entity` command.
---

## Add Entity Template Properties

| Name                                                              | Required | Description                                                                                          | Default |
| ----------------------------------------------------------------- | -------- | ---------------------------------------------------------------------------------------------------- | ------- |
| [Entities](/docs/entities)                                        | Yes      | A list of entities to create database endpoints.                                                     | _None_  |
| [AddSwaggerComments](/docs/swagger#add-swagger-comments-property) | No       | Determines whether or not Craftsman should scaffold out detailed swagger comments for each endpoint. | false   |

## Entity Template Example

An entity template is the same as a standalone as it is for a new boundary with the only difference being the `Entities` property is at the root level.
For example:

```yaml
Entities:
  - Name: Recipe
    Features:
      - Type: GetList
      - Type: GetRecord
      - Type: AddRecord
      - Type: UpdateRecord
      - Type: DeleteRecord
    Properties:
      - Name: Title
      - Name: Directions
```
