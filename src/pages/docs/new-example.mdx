---
title: New Example Command
shortTitle: New Example
description: Scaffolds out an example project via CLI prompts into the current directory.
---

```shell
craftsman new example
```

## Prompts

- **Type**: the type of example you'd like to generate.
- **Name**: the name of the folder you'd like to put the example project in
